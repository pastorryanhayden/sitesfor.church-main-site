2. Adding Basic Info and Home Page

Video Overview#

Note

Convention: If a table starts with "Instructions" it's intended to be a single row table. Only the information put in row 1 will matter and be used.

Main Church Info#

The "Main Stuff" tab controls the following:

  • Church Name: The church name as it is displayed in several parts of the site.
  • Church Tagline: Sometimes a church has a tagline "i.e. The friendly church" that they want displayed. Not all themes accept this.
  • Church Logo: This is where the church logo file is kept. Your site rep will help you get the logo file to be what it needs to be.

Service Schedule#

The "Service_Schedule" tab is where the service schedule (prominently displayed on all themes) is managed. If you need to make changes to your service schedule - do it here.

Please note - order does matter here. The order the service schedule is in Airtable will be the order it is displayed on your site.

Contact Details#

Contact information is also prominently displayed on your site. You can change it or add to it here. The Facebook, Twitter, and Youtube fields are optional and will never affect the look of your site.

Location Details (And Map)#

The "Location" tab is where you will manage your church's physical address as well as the map which is shown on the welcome page (and sometimes contact page).

To add a google map - simply go to http://maps.google.com and find your church using the search box. Then click "share" and then "embed."

Copy and paste the contents into the "Google_Maps_Embed_Code" field. Then make two changes:

  • Change where it says 'width="600"' to 'width="100%"'.
  • Change where it says 'height="450"' to 'height="400"'.

Home Page#

The last tab "Home_Page" allows you to tweak the text and main image on your home page. Notice the following:

  • Intro_Heading - The main heading on the home page.
  • Intro_Text - The paragraph below the main heading and/or the paragraph in the footer of the page.
  • Main_Image - the main hero image of the page.
